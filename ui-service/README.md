# UI Service

This directory contains the UI service for the healthcare platform, built using React and Vite.

## Structure

- `src/`: Contains the source code for the React application.
  - `App.tsx`: The main component of the application.
  - `components/`: Directory for reusable components.

- `package.json`: Lists the dependencies and scripts for the React application.

- `vite.config.ts`: Configuration file for Vite, including build settings.

## Getting Started

To get started with the UI service, follow these steps:

1. **Install Dependencies**: Run `npm install` to install the required dependencies.
2. **Run the Application**: Use `npm run dev` to start the development server.
3. **Build for Production**: Use `npm run build` to create a production build.
